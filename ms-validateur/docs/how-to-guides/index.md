# Guides pratiques

Ce dossier repond aux taches frequentes d'exploitation et de maintenance de `ms-validateur`.

Chaque guide vise une action precise et directement reproductible.

## Contenu

- `run-tests.md` : executer pytest et regenerer les rapports.
- `run-with-docker.md` : construire et lancer le service avec Docker ou Docker Compose.
- `add-sensor.md` : ajouter un nouveau capteur et ses seuils.
- `ci-cd-pipeline.md` : comprendre et utiliser le pipeline GitHub Actions.
