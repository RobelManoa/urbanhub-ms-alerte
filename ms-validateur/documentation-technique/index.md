# Documentation technique - ms-validateur

Ce dossier regroupe les six documents demandes pour la partie technique du microservice `ms-validateur`.

La documentation principale reste organisee avec la methode Diataxis dans `ms-validateur/docs/`. Les fichiers de ce dossier sont une version de restitution, plus proche du format attendu pour l'evaluation.

## Documents

- `01_presentation.md` : presentation du microservice.
- `02_pipeline.md` : pipeline CI/CD.
- `03_sonarcloud.md` : analyse SonarCloud High/Critical.
- `04_snyk.md` : analyse cybersecurite Snyk.
- `05_guide_prise_en_main.md` : guide de prise en main.
- `06_evolutivite.md` : evolutivite et integration UrbanHub.

## Lien avec Diataxis

- Presentation et evolutivite : explications.
- Pipeline, SonarCloud et Snyk : reference et explications.
- Guide de prise en main : tutoriel et guide pratique.
